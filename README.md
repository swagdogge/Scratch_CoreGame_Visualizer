This beautiful python file can be used to convert a CoreGame replay (replay.json) into a format that can be imported into my scratch CoreGame visualizer
( https://scratch.mit.edu/projects/1313810939/ )
replay.json needs to exist in the same folder as this tool when it is executed.

CoreGame is a programming challenge where you write a bot that automatically
competes in a strategy-style game against other players’ bots.
Instead of playing manually, your code controls everything—movement, resource management, and combat.
The goal is to outsmart others through better algorithms and logic.
You can find out more about it on ( https://coregame.sh/ )
